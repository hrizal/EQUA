# Equa Pro - 31-Band Professional Windows Equalizer

Equa Pro is a high-performance, studio-grade 31-band graphic equalizer for Windows. Built with a parallel DSP engine, it provides ultra-precise frequency control with zero-latency signal processing.

## 🚀 Features
- **31-Band Precision**: Full control over the 20Hz - 20kHz frequency spectrum.
- **Parametric Hybrid**: Includes High-Pass Filter (HPF) and Low-Pass Filter (LPF).
- **Intelligent Curve Mode**: Create smooth EQ curves by moving a single slider.
- **Pre-Amp Gain**: Boost your signal up to 2000% (+26dB) for quiet sources.
- **Studio-Grade Presets**: Save and load your favorite sound profiles.
- **Minimalist Aesthetic**: Premium dark-mode UI designed for professional focus.

## 🛠 Prerequisites
To use Equa Pro effectively, you need a virtual audio bridge to route your system sound through the equalizer:
1. **VB-Audio Virtual Cable** (Recommended): [Download Here](https://vb-audio.com/Cable/)
2. **WebView2 Runtime**: Most Windows 10/11 systems have this built-in. If not, [get it here](https://developer.microsoft.com/en-us/microsoft-edge/webview2/).

## 📖 How to Use
1. **Set Default Output**: In Windows Sound Settings, set your Default Playback Device to **CABLE Input**.
2. **Open Equa Pro**: Select **CABLE Output** as the **INPUT SOURCE**.
3. **Select Speakers**: Select your actual **Headphones/Speakers** as the **OUTPUT DEVICE**.
4. **Power On**: Click the **Red Power Button** to activate the real-time DSP bridge.
5. **Tune**: Adjust the sliders to achieve your perfect sound.

## 📦 Installation
Just download `equa.exe` from the latest release and run it. No installation required (Portable).
